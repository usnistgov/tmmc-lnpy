lnpy.xGrandCanonical.betaF_alt¶

xGrandCanonical.betaF_alt(betaF_can, correction=True)[source]¶

Alternate scaled Helmholtz free energy \(\beta \overline{F}\).

Calculated using

\[\beta \overline{F} = \sum_N [\Pi(N) \beta F(N) + C(N)]\]
Parameters

  • betaF_can (array-like) – Value of \(F(N)\)

  • correction (bool, default True) – If True, \(C(N) = \ln \Pi(N) \).
