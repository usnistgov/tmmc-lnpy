lnpy.lnPiMasked.new_like¶

lnPiMasked.new_like(lnz=None, base=None, mask=None, copy=False)[source]¶

Create new object with optional parameters

Parameters

  • lnz (float or sequence of floats) – Value(s) of lnz (log of activity)

  • base (Data) – lnPi data object.

  • mask (bool or array-like, optional) – Where mask is True, values are “masked out”.

  • copy (bool, optional) – If True, copy the data. If False, attempt to use view.
