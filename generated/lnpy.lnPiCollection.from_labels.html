lnpy.lnPiCollection.from_labels¶

classmethod lnPiCollection.from_labels(ref, labels, lnzs, features=None, include_boundary=False, labels_kws=None, check_features=True, **kwargs)[source]¶

Create from reference lnPiMasked and labels array

Parameters

  • ref (lnPiMasked)

  • labels (sequence of label arrays) – Each labels[i] will be used to construct multiple phases from single (reweighted) \(ln \Pi(N)\)

  • lnzs (sequence) – Each lnzs[i] will be passed to ref.reweight.

  • features (int, optional)

  • include_boundary (bool, default False)

  • labels_kws (mapping, optional)

  • check_features (bool, default = True)

  • **kwargs – Extra arguments past to from_list()

labels_to_masks, from_list
